# TDSMH_P1P6_VMG

## Descripción
Aplicación Android desarrollada en **Java** con **Android Studio** que implementa un formulario de captura de datos en dos pantallas:

1.  **Primera pantalla (Formulario):** Solicita al usuario su **nombre**, **edad**, **correo electrónico** y **teléfono**.
2.  **Segunda pantalla (Resultados):** Muestra todos los datos ingresados y, basado en la edad, presenta una alerta indicando si el usuario puede o no consumir alcohol (mayor o menor de 20 años).

## Cómo ejecutar el proyecto
1. Clona el repositorio:
   
   ```bash
   git clone https://github.com/valmejia/TDSMH_P1P6_VMG.git

2. Abre Android Studio.
3. Espera a que se sincronicen las dependencias (Gradle).
4. Conecta un dispositivo físico o inicia un emulador.
5. Haz clic en Run.
